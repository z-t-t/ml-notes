一个具体的神经网络的向量化代码的实现（各数据如下图所示）：
![alt text](fc88a24b1b9fa15ded12665a2e69af4.png)
```python
import numpy as np
AT=np.array([[200,17]])
W=np.array([
    [1,-3,5],
    [2,4,-6],
    ])
b=np.array([[-1,1,2]])
def dense(AT,W,b):
    Z=np.matmul(AT,W)+b
    A_out=1/(1+np.exp(-Z))
    return A_out

print(dense(AT,W,b)) #[[1.00000000e+000 2.45261912e-231 1.00000000e+000]]
```
